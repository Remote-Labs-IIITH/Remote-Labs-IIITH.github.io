How to use this dashboard

  • Overview
  • Set up the experiment by setting up the angle using the slider, this sets the servo angle which controls the position of the wire. The wireframe will adjust it's position in front of the laser to keep the selected wire in front of the light source.
  • After setting up the angle, click on Reload Plot to check the changes in the interactive plot.
  • Make measurements using your mouse pointer by noting down the coordinates of the minimas and central maxima and calculate the distances.
  • Using these distances, you can use the formula provided in the theory page to estimate the wire width.
  • Analysis
  • True widths of the wires are:
  • when angle is around 5, it corresponds to a wire of width 0.40mm
  • when angle is around 80, it corresponds to a wire of width 0.27mm
  • when angle is around 150, it corresponds to a wire of width 0.20mm
  • Compare the wire width that you got with the true value. Perform this experiment atleast 5 times. Calculate error, mean and standard deviation of your values. Report the error.
